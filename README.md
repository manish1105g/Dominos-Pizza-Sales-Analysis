# 🍕 Pizza Sales Analysis Using SQL

## 📌 Project Overview

This project focuses on analyzing pizza sales data using SQL to uncover valuable business insights related to orders, revenue, pizza categories, customer preferences, and sales trends. The analysis helps in understanding business performance and customer ordering behavior through structured SQL queries.

## 🎯 Objectives

* Analyze total pizza sales and revenue generation
* Identify top-selling pizzas and popular sizes
* Understand customer ordering patterns
* Perform category-wise and time-based sales analysis
* Generate insights for better business decision-making

## 📊 Project Features

### 🍕 Sales Analysis

* Total number of orders placed
* Total revenue generated from pizza sales
* Average pizzas ordered per day

### 🏆 Top Performing Pizzas

* Top 5 most ordered pizza types
* Highest-priced pizza identification
* Most common pizza size ordered

### 📂 Category Analysis

* Category-wise pizza distribution
* Quantity sold by pizza category
* Revenue contribution by pizza type

### ⏰ Time-Based Analysis

* Distribution of orders by hour
* Daily ordering trends
* Cumulative revenue analysis over time

### 💰 Revenue Insights

* Revenue generated by each pizza type
* Percentage contribution to total revenue
* Sales performance tracking
